Overview

**Core Modules**: High-level Python interfaces for common operations

**NiFi APIs**: Complete generated client for all NiFi REST endpoints

**NiFi Models**: Data structures and DTOs used by NiFi APIs

**Registry APIs**: Complete generated client for NiFi Registry

**Registry Models**: Data structures used by Registry APIs
